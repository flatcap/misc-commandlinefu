Find all php files in a directory and sub-directories even if they don't have an extension.

to limit to current directory use

find . -maxdepth 1 -exec grep -q '<?php' {} /dev/null \; -ls

for command line scripts, could also use

find . -exec grep -q '#!/usr/bin/php' {} /dev/null \; -ls

this would also work for bash scripts

find . -exec grep -q '#!/bin/bash' {} /dev/null \; -ls
