create missing md5 for all files in directory
find . ! -name \*.md5 -exec 'md5sum "{}" > "{}".md5' \;

Just use find. No need to test file existence. On gnu find you can limit directory depth. Use "{}" to manage correctly files with spaces

create missing md5 for all files in directory
for f in $(ls | grep -v [.md5]$); do if [ -f $f ] && [ ! -f $f".md5" ]; then echo "missing md5 for '${f}' will now create..."; md5sum $f > $f".md5"; echo "created"; fi; done;

creates a .md5 file for each file in a directory that doesn't have a [filename].md5 file, but doesn't create for md5 files or directories.
