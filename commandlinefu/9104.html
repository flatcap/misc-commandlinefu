Not sure how to determine the default font, but you can use other typefaces.

convert -list font |grep Font:

will give you the list of names you can use with "-font"

I strongly recommend using a monospaced font, like FreeMono-Medium or Liberation-Mono-Regular for command output.

Can be useful for making a "screenshot" of what a website looks like in text-only mode:

/usr/bin/lynx -dump -nolist http://www.commandlinefu.com/ | /usr/bin/convert -font "FreeMono-Medium" label:@- output.png
