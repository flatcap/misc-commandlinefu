Extract a bash function

sed -n '/^function h\(\)/,/^}/p' script.sh 
$ sed -n '/^function h\(\)/,/^}/p' .bash_profile
function h() {
    if [ -z "$1" ]
    then
	history
    else
	history | grep "$@"
    fi
}

I often need to extract a function from a bash script and this command will do it.
