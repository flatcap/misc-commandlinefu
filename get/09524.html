Get the time and date of the last server reboot

date -d "$(uptime | awk '{gsub(/,/,"",$3);gsub(/:/," hours ",$3); print "- " $3 " minutes"}')"

Alternatives

last reboot
last reboot 
reboot   system boot  2.6.40.4-5.fc15. Fri Sep 30 10:40 - 09:41 (19+23:00)

- shorter

- more readable

What others think

You can also find the time in the output of:

last reboot | head -1

and the less portable (Linux only):

date -d "$(awk '{print $1 " seconds ago"}' /proc/uptime)"
