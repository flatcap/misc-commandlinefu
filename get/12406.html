Check if a string is into a variable

[[ ${var##*yourstring*} != ${var} ]] 
$ var=foobar
$ [[ ${var##*foo*} != ${var} ]]
$ echo $?
0
$ [[ ${var##*notfoo*} != ${var} ]]
$ echo $?
1

Returns true (0) if the string is into $var, or false (1) if not.
