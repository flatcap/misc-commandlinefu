Convert a string to "Title Case"

echo 'This is a TEST' | sed 's/[^ ]\+/\L\u&/g' 
This Is A Test

Alternatives

echo "this is a test" | sed 's/.*/\L&/; s/[a-z]*/\u&/g' 
$ echo "this is a test" | sed 's/.*/\L&/; s/[a-z]*/\u&/g'
This Is A Test
$

Converts the first letter of each word to upper case

python -c "print 'this is a test'.title()"
