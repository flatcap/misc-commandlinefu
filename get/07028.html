^^ will do all characters, comma and double-comma will covert to lowercase. ~ toggles the first character of each word, ~~ toggles all characters. To title case a string:

string='mIXed uP CAse' string=${string,,} string=${string~} echo "$string"

By the way, all of this requires Bash 4.
