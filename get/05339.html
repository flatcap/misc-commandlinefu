Unix commandline history substitution like ^foo^bar BUT for multiple replacements

!!:gs/Original/New/ 
# groovy extractCalls.groovy 20100407 && groovy extractInvokes.groovy 20100408 && groovy extractInvokes.groovy 20100409
# !!:gs/extractInvokes.groovy/extractCallbacks.groovy/
# groovy extractCallbacks.groovy 20100407 && groovy extractCallbacks.groovy 20100408 && groovy extractCallbacks.groovy 20100409

#1 You invoked a command chain.

#2 You globaly replace the scriptnames and execute them direct

#3 The new executed command chain

[# ^foo^bar] would only replace the firts match [foo] with [bar].
