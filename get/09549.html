Run a command when a file is changed

while inotifywait -e modify /tmp/myfile; do firefox; done

Alternatives

while :; do n=$(md5 myfile); if [ "$h" != "$n" ]; then h=$n; scp myfile myserver:mydir/myfile; fi; sleep 1; done
