Edit a file on a remote host using vim

vim scp://username@host//path/to/somefile

What others think

Trivial variation, this also works with vimdiff

e.g.

vimdiff /etc/fstab scp://user@remote//etc/fstab

which is nice.

This does not work on all versions of vim. For example:

VIM - Vi IMproved 7.2 (2008 Aug 9, compiled May 29 2010 04:33:52)
