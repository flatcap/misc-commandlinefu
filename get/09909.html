list with full path

printf "$PWD/%s\n" * 
$ touch apple banana cherry
$ printf "$PWD/%s\n" *
/home/flatcap/apple
/home/flatcap/banana
/home/flatcap/cherry

List the full path of some files.

You can add ".*" on the end if you want to display hidden files.

Alternatives

ls -d $PWD/*
ls -d1 $PWD/{.*,*}
find $(pwd) -maxdepth 1 -name "*" -printf "%p\n"
find `pwd` -maxdepth 1 -exec ls --color -d {} \;
ls -d1 $PWD/*
