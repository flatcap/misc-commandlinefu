Replace Space In Filenames With Underscore

for file in "* *"; do mv "${file}" "${file// /_}"; done

@ http://sgowtham.net/blog/2007/10/18/bash-replace-space-in-filenames-with-underscore/

Alternatives

rename 's/ /_/g' * 
for file in *; do echo mv "${file}" "${file// /_}"; done

rename is often an alias to prename, bundled with perl.
