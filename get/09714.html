Redirect tar extract to another directory

tar xfz filename.tar.gz -C PathToDirectory

This command can also be written in a more compact way:

tar zxfC filename.tar.gz Path

You only need to pay attention to the order of "fC" switches, they should match the order of the given arguments (filename.tar.gz, Path).
