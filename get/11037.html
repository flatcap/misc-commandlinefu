Show permissions of current directory and all directories upwards to /

namei -m $(pwd) 
f: /home/foo/bar
 drwxr-xr-x /
 drwxr-xr-x home
 drwxr-xr-x foo
 drwxr-xr-x bar

NB not 'namei -m .', as it slices the path you give it.

Alternatives

dir=$(pwd); while [ ! -z "$dir" ]; do ls -ld "$dir"; dir=${dir%/*}; done; ls -ld /
