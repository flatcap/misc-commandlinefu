Your command actually connects to the most RECENTLY created session, because screen lists them in reverse order.

If you actually wanted the FIRST created, alter your awk to something like:

screen -x $(screen -ls | awk '/^\t/{ x=$1 }; END { print x }')

Alternatively, you can find the most recently unused (detached) screen:

screen -x $(screen -ls | awk '/(Detached)/{x=$1;exit} END { print x }')
