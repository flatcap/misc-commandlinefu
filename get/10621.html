Rename files with unique, randomly generated file names

for i in *.txt; do j=`mktemp | awk -F. '{print $2".txt"}'`; mv "$i" "$j"; done

Hmm... It works, but it's a bit messy.

First of all, stop mktemp creating files that we don't use:

mktemp -u

Then use bash to replace awk (strip out everything (*) until dot):

${filename##*.}.txt

That leaves us with:

for i in *.txt; do F=$(mktemp -u); mv "$i" ${F##*.}.txt; done
