xargs for builtin bash commands

xargsb() { while read -r cmd; do ${@//'{}'/$cmd}; done; }
xargs for builtin bash commands

Similar to xargs -i, but works with builtin bash commands (rather than running "bash -c ..." through xargs)

Alternatives

bargs { while read i; do "$@" "$i"; done }
